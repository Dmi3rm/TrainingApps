# Nine
The application is the analogue of Specks game, but the game field has 3x3 size. Press the Start button while the field is empty, so the program will randomly arrange the checkers and start to move them in the correct order. You may also set start values by yourself, pressing on the cells and entering the values from 1 to 8 using keyboard (press Start button after). 
---
Программа является аналогом игры "Пятнашки", только с полем 3х3. При пустых клетках нажмите старт, программа рандомно расставит шашечки и начнет собирать их в правильном порядке. Если желаете сами расставить шашечки, нажимайте на соответствующие клетки и вводите в них значения от 1 до 8 (одна должна остаться пустой); затем нажмите старт.
